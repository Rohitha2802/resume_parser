This project is a simple Resume Parser built using Python.

It takes a plain text resume (resume.txt) and extracts structured information such as:
1. Name
2. Email
3. Phone Number
4. Skills
5. Experience
6. Education
7. About Me (Summary)

TECHNOLOGIES USED:
1. Python
2. Regular Expressions (re module)
3. JSON



HOW TO RUN:
  Clone the repository:
  
  git clone https://github.com/Rohitha2802/resume_parser.git
  
  cd resume_parser
  
  Run the script:
  
  python extract_parse.py
